프로그래밍(programming)
: 목적에 맞게 프로그램을 작성하는 과정을 의미
: 알고리즘 이나 프로그래밍 언어로 사용이 된다.

C언어 실행 파일을 생성하는 순서
소스파일 -> 선행처리기 -> 컴파일러 -> 오브젝트 파일
-> 링커(시동 코드, 표준 라이브러리 파일)-> 실행파일

source file
: c언어의 문법에 맞게 논리적으로 작성된 프로그램ㅇ
  원시 파일 or 소스 파일
: 확장자 .c

preprocessor
: #로 시작하는 선행처리 지시문의 처리 작업
: 컴파일러가 작업하기 좋도록 소스를 재구성해주는 역할

compiler
: 기계어만을 이해할 수 있음.
: 컴퓨터가 이해할 수 있도록 변환 시키는 것.
: 오브젝트 파일은 기계어로 변환된 파일
: 확장자 .o .obj

linker
: 인터페이스를 담당하는 시동코드(start-up code)가 없음.
: 여러 개의 소스파일을 작성하여 
  최종적으로 링크를 통해 하나의 실행 파일로 만드는 것
  분항 컴파일이라고 함.

executable file
: 위와 같은 과정을 거쳐 실행 파일을 변환
: 소스파일에서 한 번에 실행 파일로 생성해주는 것
: .exe
