C 언어 개요
C언어는 시스템에서 사용할 수 있는 프로그래밍 언어

운영체제의 커널은 C언어를 통해 구현되어 있음

C언어는 시스템 프로그래밍에 어울리지만, 
        응용 프로그래밍에도 많이 사용되는 프로그래밍 언어이다.


C 언어란?
컴퓨터 시스템에서 사용할 수 있는 프로그래밍 언어

저급언어 -> machine language , assembly language
: 주의점 -> 실행 속도는 빠르지만, 프로그램의 유지보수는 힘듬.

고급언어 -> 컴파일러나 인터프리터에 의해 
           기계가 이해할 수 있는 언어로 번역된다.
주의점 -> 실행 속도는 느리다,

// C언어는 정차 지향 프로그래밍 언어
   (procedure-oriented programming language)


C 언어의 역사
: 유닉스(UNIX) 운영체제에서 사용하기 위한
  범용적인 고급 언어
: A언어 , B언어 , C언어 


C언어의 탄생 배경
: 어셈블리어는 하드웨어에 종속적인 언어
: 다른 하드웨어로 이식할 수 없었음.

// 이식성이 좋고, 쉬운 언어가 필요했었음.
// 시스템 프로그래밍에 어울리지만, 
          응용 프로그래밍에도 많이 사용되는 언어.


C언어의 표준
: ANSI에서 표준화 작업을 진행해왔었음.
: ISO/IEC에서 표준화를 담당하고 있음.
: C언어의 최신 표준은 ISO/IEC 9899:2011 이고, C11
