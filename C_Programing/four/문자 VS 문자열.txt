기본적인 입출력

1. 기본적인 입출력
: 기억장치에 저장되는 파일사용하는 것과 
  비슷하게 입출력 장치를 다룬다.
: 콘솔장치(키보드,모니터) 같이 
  C 프로그램에서 자동으로 열리는 파일처럼 다뤄진다
-> C언에서 stdin표준스트립을 통해 입력장치를 다룬다.
          stdout표준스트립을 통해 출력방치를 다룬다. 


2. 스트림(stream)
-> C프로그램은 콘솔의 입출력을 직접 다루지 않고,
   스트림(stream)이라는 것을 통해 다룬다.
: 입력이나 출력이 표현된 데이터의 이상화된 흐름을 의미한다.
// 운영체제에 의해 생성되는 가상의 연결 고리를 의미

콘솔 장치에 대한 스트림은 프로그램 실행 시
자동으로 생성되며, 프로그램 종료 시 자동으로 소멸한다.

표준 스트림(standard stream)
stdin -> 표준 입력 스트림
stdout -> 표준 출력 스트림
stderr -> 표준 오류 스트림


3. EOF(End OF File)
-> 운영체제에서 파일의 끝을 탐지하는 방법는 
   각 운영체제 마다 다릅니다.
-> C언어는 운영채제와 상관없이, 파일의 끝은 특별한 값을 
   반환하도록 한다.
유닉스(UNIX)시스템에서는 Ctrl + D
window 명령창에서는 Ctrl + Z 를 누르고 Enter

#include <stdio.h>
int main(void){
  char ch;

  while((ch = getchar()) != EOF){
    putchar(ch)
  }
  return 0;
}
