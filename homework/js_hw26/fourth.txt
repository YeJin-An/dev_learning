# Interaction: alert, prompt, confirm

# alert
1> 메시지를 표시하고, 
   사용자가 "확인"을 누를 때까지 기다림,
2> 메세지가 있는 창 [모달창]

# prompt
1>   title -> 방문제에게 표시할 텍스트
2> default -> 선택적 두 번째 매개변수,
              입력 필드의 초기값
3> 2번때 매개변수는 선택사항이나
   제공하지않으면 undefuned를 프롬프트에 삽입

# contirm
1> question확인 및 취소라는 
   두 개의 버튼이 있는 모달창 표시
2> 결과는 true확인을 누른 경우, false 그렇지 않는 경우

alert, prompt, confirm 정리
1> alert    메시지를 보여준다.
2> prompt   택스트 입력하라는 메시지 표시
3> contirm  사용자가 '확인' 또는 '취소'

모달 창의 정확한 위치는 브라우저에 의해 결정됩니다. 
일반적으로 중앙에 있습니다.
창의 정확한 모양은 브라우저에 따라 다릅니다. 
수정할 수 없습니다.
