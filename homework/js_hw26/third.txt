# Data tyes

let message = "hello';
message = 123456;
> 이것을 허용하는 프로그래밍 언어를 
  '동적 유형' -> 어느 것에도 구속되지 않는다는 의미

# Numbers
1> Infinity -> 수학적 무한대의 의미
2>      NaN -> 계싼 오류가
3>   BigInt -> 임의의 길의의 정수를 나타내기 위함.

# String
1> "" , '' , ``
2> ${...} -> 변수와 표현식을 문자열에 포함
3> char타입이 없음.
4> 하나의 문자 or 많은 문자로 구성될 수 있음

# Boolean
1> true 와 false

# Null
1> '아무것도 없음'
2> '비어 있음'
3> '알수 없는 값'

# undefined
1> 변수가 선언 되었지만, 할당되지 않아 undefined
2> 기술적 undefined로 변수에 명시적으로 할당가능

# Objects AND Symbols
1> 데이터 컬렉션과 더 복잡한 엔터티를 저장하는데 사용
2> 객체에 대한 고유 식별자를 만드는 데 사용

# typeof 연산자
1> 인수의 형태를 돌려줌.
2> 다른 유형의 값을 다르게 처리
   확인하려는 경우에 유용
3> typeof(x) == typeof x

Data tyes의 정의
1> number 정수 또는 부동 소수점
2> BigInt 임의 길의 정수에 대한 것
3> string 0가 이상의 문자를 가질 수 있음
4> boolena 은 참과 거짓
5> null 알수 없는 값의 경우 - 단일 값을 갖는 독립형 유형
6> undefined 할당되지 않는 값의 경우 - 독립형 유형
6> odject 더 복잡한 데이터 구조의 경우
7> symbol 고유 식별자의 경우
